# Graphify install reference

Source: https://github.com/safishamsi/graphify

Recommended install:

```bash
uv tool install graphifyy
```

Alternatives:

```bash
pipx install graphifyy
python3 -m pip install --user graphifyy
```

Project skill install command from upstream:

```bash
graphify install --project --platform codex
```

The official Python package is `graphifyy` (double-y). The CLI command remains `graphify`.
